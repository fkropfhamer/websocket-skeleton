# Websocket-Skeleton
## required: `nodejs` and `npm`
## to start:
## 1. `npm install`
## 2. `npm start`
## 3. open Browser go to `localhost:8080`


### `INSTALL: npm install`
### `DEV SERVER: npm run dev`
### `START: npm start`
    
### `BUILD SERVER: npm run build:server`

### `DEV FRONTEND: npm run watch`
### `BUILD FRONTEND: npm run build`

### `reachable: localhost:8080`
